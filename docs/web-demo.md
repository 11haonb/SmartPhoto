# Web Demo 使用指南

Web Demo 是一个手机外框样式的单页应用，用于端到端测试 SmartPhoto 的所有功能。它直接调用 FastAPI 后端 API，不依赖 Flutter SDK。

## 启动

```bash
cd web-demo
node serve.js
# 打开 http://localhost:3000
```

如果通过 SSH 远程访问，需要端口转发：

```bash
ssh -L 3000:localhost:3000 -L 28000:localhost:28000 user@server
```

### 首页

- 展示 4 个核心功能介绍
- 点击「选择照片开始整理」进入上传页

### 相册页

- 点击上传区域选择本地图片
- 支持多选（最多 100 张）
- 选择后显示照片网格预览
- 点击「上传并开始整理」开始上传
- 显示上传进度条

### 整理进度页

- 圆环进度指示器（0% → 100%）
- 5 个阶段状态列表
- 每 3 秒自动轮询后端进度
- 阶段状态：⏳ 等待 → 🔄 处理中 → ✅ 完成

### 结果页（4 个 Tab）

| Tab | 内容 |
|-----|------|
| **时间线** | 照片按拍摄日期分组，网格展示 |
| **分类** | 按 AI 分类结果分组（人物/风景/美食...），显示数量 |
| **问题图** | 标记为模糊/过曝/欠曝/截图的照片列表 |
| **最佳照片** | 相似组展示，最佳照片高亮，其他照片半透明 |

### 设置页

- 4 个 AI 引擎选择卡片
- API Key 输入框（需要时显示）
- 退出登录

## 技术实现

Web Demo 使用原生 HTML/CSS/JS，零依赖：

| 文件 | 功能 |
|------|------|
| `index.html` | 入口 + 手机外框容器 |
| `css/phone-frame.css` | iPhone 样式外框（圆角/刘海/底部条） |
| `css/app.css` | App 内部所有样式 |
| `js/api.js` | API 客户端（12 个端点 + JWT 管理） |
| `js/auth.js` | 登录页逻辑 |
| `js/album.js` | 照片选择 + 上传 |
| `js/organize.js` | 进度轮询 + UI 更新 |
| `js/results.js` | 4-Tab 结果展示 |
| `js/app.js` | 路由 + 首页 + 设置页 + Tab 栏 |
| `serve.js` | Node.js 静态文件服务器 |

## 自定义 API 地址

默认连接 `http://localhost:28000`。如果后端在其他地址，在浏览器控制台设置：

```javascript
window.SMARTPHOTO_API_URL = 'http://your-server:28000/api/v1'
```
